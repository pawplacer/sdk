# Security Policy

## Supported Versions

Security fixes are applied to the latest published version of `pawplacer-sdk`.

## Reporting a Vulnerability

Please do not open public GitHub issues for suspected security vulnerabilities.

Email security reports to:

```text
contact@pawplacer.com
```

Include:

- A description of the issue
- Steps to reproduce, if available
- Affected versions
- Any relevant logs, payloads, or proof of concept details

We will acknowledge valid reports as soon as practical and coordinate a fix before public disclosure.

## API Keys

The SDK is intended for server-side use. Do not ship PawPlacer API keys in browser bundles, mobile apps, or other public clients.
